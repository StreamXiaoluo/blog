## 02 快捷键

### shift家族

1. shift + 移动 ：水平竖直移动
2. shift + 放大 ：等比例放大
3. shift + 旋转 ： 15度旋转

### ctrl家族

1. ctrl + 滚轮 ： 快速放大缩小
2. ctrl + 拖拽 ： 拖拽复制
3. ctrl + D （cmd + d）：快速复制，可以**复制空间和距离**
4. ctrl + M / 回车 ： 新建页面
5. ctrl + G / cmd+alt（option）： 组合
6. ctrl + S /cmd + s ： 保存
7. ctrl + Z 、ctrl + Y

### shift+ctrl

1. ctrl + shift + 移动鼠标： 水平竖直移动复制
2. ctrl + shift + alt + 放大缩小 ： 等比例中心放大缩小
3. ctrl + shift + C/V :  格式刷
4. ctrl + shift + G ： 解除组合

### 文本框操作
> Mac 系统中文本框操作使用cmd替代ctrl

1. ctrl + shift + < / >  : 快速放大缩小文字
2. ctrl + U I B 
3. ctrl + E R L

### 文字图片快速操作

1. Alt → N X H ： 插入文字
2. Alt → N P D ： 插入图片
